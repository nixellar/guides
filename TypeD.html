Introduction guide:

Type D : [Nixellar] into [NIX]

Users need to return the 'Nixellar' they hold to the issuing account (address=G...i) on the Stellar blockchain, so that we may remove them from circulation. This is a relatively cheaper process.
  1. User needs to communicate their "NIX Platform address" to recieve 'NIX'.
  2. User has to send their 'Nixellar' to "Nixellar ('NIX') issuing acount, G...i.
  3. We send the 'NIX' to the users "NIX Platform address".
